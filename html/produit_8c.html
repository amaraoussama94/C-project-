Adds a new product to the database interactively.

void lister_produits_interactif ( sqlite3 * db)

Affiche la liste des produits de manière interactive.

Affiche tous les produits de la base de données.

Cette fonction appelle db_lister_produits pour afficher tous les produits présents dans la base de données SQLite passée en paramètre. Si une erreur survient lors de l'affichage, un message d'erreur est affiché à l'utilisateur.

Parameters
dbPointeur vers la base de données SQLite.

◆ modifier_produit_interactif()

void modifier_produit_interactif ( sqlite3 * db)

Interactively modifies the details of a product in the database.

Demande à l'utilisateur un ID et les nouvelles infos, puis modifie le produit.

Prompts the user to enter the ID of the product to modify, then asks for the new name, quantity, and price. Updates the product in the database using the provided information. Displays a success or error message based on the outcome.

Parameters
dbPointer to the opened SQLite database connection.

◆ supprimer_produit_interactif()

void supprimer_produit_interactif ( sqlite3 * db)

Supprime un produit de la base de données de manière interactive.

Demande à l'utilisateur un ID et supprime le produit correspondant.

Cette fonction demande à l'utilisateur de saisir l'identifiant (ID) du produit à supprimer, puis appelle la fonction db_supprimer_produit pour effectuer la suppression dans la base de données SQLite. Elle affiche un message indiquant si la suppression a réussi ou échoué.

Parameters
dbPointeur vers la base de données SQLite ouverte.
