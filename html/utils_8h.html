Cette fonction lit une ligne de texte, supprime le caractère '

Lit un entier positif depuis l'entrée standard avec validation.

This function reads a line from stdin, removes the trailing newline character, and checks if the input consists only of digit characters. If the input is valid, it converts the string to an integer and returns it. If the input is invalid or reading fails, it returns -1.

Returns
The integer value read from input, or -1 if input is invalid or reading fails.

Lit un flottant positif depuis l'entrée standard avec validation.

This function prompts the user to enter a floating-point number as a string, validates the input to ensure it contains only digits and at most one decimal point, and then converts it to a float. If the input is invalid or reading fails, the function returns -1.0f.

Returns
The floating-point value entered by the user, or -1.0f if input is invalid.
