void pause_console ( )

Pauses the console and waits for the user to press a key before continuing.

This function displays a prompt message asking the user to press any key to return to the menu. It uses platform-specific commands to wait for a key press:

  • On Windows systems, it uses the "pause" command with output redirected to null.
  • On other systems (e.g., Linux, macOS), it uses the "read" command to wait for a single key press.
